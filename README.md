# 📘 Lógica de Programação — Pro-Profissão

Este repositório contém exercícios e práticas realizadas durante meus estudos de **Lógica de Programação no Instituto PROA**.

Atualmente estou aprendendo **JavaScript** e conceitos fundamentais de programação, como entrada de dados, estruturas condicionais e resolução de problemas utilizando lógica.

## 🎯 Objetivo

Desenvolver habilidades de **lógica de programação** e aprender a estruturar soluções utilizando **JavaScript**, criando pequenos projetos e exercícios práticos.

## 🧠 Conteúdos estudados

Durante os estudos estou praticando:

* 📥 Entrada de dados
* 📤 Saída de dados
* 🔢 Variáveis
* ➕ Operadores matemáticos
* ⚖️ Operadores lógicos
* 🔀 Estruturas condicionais (`if`, `else`, `else if`)
* 🌐 Integração entre **HTML e JavaScript**
* 🧩 Resolução de problemas com programação

## 💻 Tecnologias utilizadas

* **JavaScript**
* **HTML**
* **VS Code**

## 📂 Estrutura do repositório

Os exercícios são organizados em pastas. Cada pasta representa uma atividade ou prática.

Dentro de cada pasta existem dois arquivos:

* um arquivo **HTML** (interface)
* um arquivo **JavaScript** (lógica do programa)

Exemplo:

```bash
logica-programacao-proa/

├── codar1/
│   ├── hora-de-codar1.html
│   └── hora-de-codar1.js
│
├── codar2/
│   ├── hora-de-codar2.html
│   └── hora-de-codar2.js
│
└── README.md
```

## ▶️ Como executar os exercícios

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/logica-programacao-proa.git
```

2. Abra a pasta do projeto.

3. Abra o arquivo **.html** no navegador.

A interface HTML permitirá inserir dados e executar a lógica programada em **JavaScript**.
